Sổ tay Hướng dẫn

✂️ 1. Chế độ Cắt PDF

- Tách tất cả: Mỗi trang của file gốc sẽ trở thành 1 file PDF riêng biệt.

- Chọn trang: Gộp các trang chỉ định vào 1 file PDF duy nhất. Ví dụ 1, 3-5 → 1 file chứa trang 1, 3, 4, 5.

- Chia đều: Cắt tài liệu thành các phần có số trang bằng nhau. Ví dụ file 10 trang, nhập chia 2 => thành 5 file.

- Cắt nhiều file: Mỗi nhóm trang (phân cách bởi dấu phẩy) sẽ tạo thành 1 file PDF riêng. Ví dụ 1-3, 5-6 → File 1 chứa trang 1÷3, File 2 chứa trang 5÷6.

📚 2. Chế độ Ghép PDF

Chọn 1 file gốc làm nền tảng, sau đó tải thêm các file khác. Bạn có thể chọn chèn lên đầu, dưới cùng, hoặc chèn vào giữa (sau trang số X) của file gốc.

🖼️ 3. Chế độ Ảnh -> PDF

Chọn nhiều tấm ảnh (chụp bài làm, tài liệu). Công cụ sẽ tự động căn giữa, co giãn và đóng thành 1 quyển PDF chuẩn kích thước giấy A4 tuyệt đẹp.

🛠️ 4. Chế độ Sửa trang

- Xóa: Loại bỏ các trang trắng hoặc bị lỗi bằng cách nhập số trang (VD: 2, 4).

- Xoay: Sửa các trang scan bị ngược. Nhập trang cần xoay và chọn góc xoay 90/180/270 độ.

©️ 5. Chế độ Đóng dấu

In chữ mờ bản quyền lên các trang PDF với nhiều tùy chỉnh:

- Nội dung: Nhập bất kỳ văn bản nào, ví dụ THẦY LỰC - THPT NGUYỄN HỮU CẢNH.

- Màu sắc: Chọn từ bảng màu có sẵn (đỏ, xanh, tím...) hoặc dùng bộ chọn màu tùy ý.

- Độ mờ & Cỡ chữ: Kéo thanh trượt để điều chỉnh phù hợp với từng loại tài liệu.

- Kiểu đóng dấu: Chéo giữa (xoay 45°), Ngang giữa (nằm ngang), hoặc Lát gạch (trải đều 9 vị trí trên trang).

- Áp dụng trang: Tất cả trang, chỉ trang lẻ, chỉ trang chẵn, hoặc chỉ định trang cụ thể (VD: 1, 3-5).

🔒 6. Chế độ Bảo mật

Gồm 2 công cụ bảo vệ tài liệu độc lập:

- Chống sao chép văn bản: Chuyển đổi toàn bộ nội dung PDF thành ảnh nhúng — người xem không thể bôi đen, sao chép hay tìm kiếm văn bản. File trông y hệt bản gốc. Có 3 mức chất lượng ảnh: Nhỏ (file nhẹ), Chuẩn (khuyên dùng), Cao (sắc nét nhất). Lưu ý: file kết quả sẽ lớn hơn bản gốc.

- Xóa thông tin ẩn (Metadata): Loại bỏ các thông tin nhúng trong file PDF như tên tác giả, tổ chức, phần mềm tạo file, ngày tạo & chỉnh sửa, tiêu đề tài liệu. Hữu ích khi cần chia sẻ tài liệu mà không muốn lộ danh tính hay nguồn gốc.

🔢 7. Chế độ Số trang

Tự động thêm số trang vào mỗi trang PDF với đầy đủ tùy chọn:

- Vị trí: Cuối trang hoặc đầu trang.

- Căn lề: Trái, giữa, hoặc phải.

- Định dạng: Chỉ số (1 2 3), có chữ (Trang 1), hoặc phân số (1/10).

- Bắt đầu từ: Có thể đặt số bắt đầu tùy ý, ví dụ bắt đầu từ 5 nếu tài liệu là phần 2.

- Màu sắc & Cỡ chữ: Chọn màu phù hợp với tài liệu.

📷 8. Chế độ Xuất ảnh

Chuyển từng trang PDF thành file ảnh, tất cả được nén vào 1 file ZIP để tải về.

- Định dạng: JPEG (file nhỏ, phù hợp chia sẻ) hoặc PNG (rõ nét tuyệt đối, nền trong suốt).

- Độ phân giải: ×1 (nhỏ), ×2 (chuẩn — khuyên dùng), ×3 (cực nét, file lớn).

- Chọn trang: Xuất tất cả hoặc chỉ định trang cụ thể (VD: 1-3, 5).

Hữu ích khi cần dùng ảnh từ PDF để chèn vào Word, PowerPoint, hoặc đăng lên mạng.

🗜️ 9. Chế độ Nén PDF

Giảm dung lượng file PDF để dễ dàng gửi email, upload lên hệ thống quản lý hoặc chia sẻ qua Zalo/Messenger.

- Nhẹ: Giảm ~30–50%, chất lượng gần như nguyên bản.

- Chuẩn: Giảm ~60–70%, cân bằng tốt giữa chất lượng và dung lượng (khuyên dùng).

- Mạnh: Giảm ~80%+, file rất nhỏ, phù hợp khi chỉ cần đọc, không cần in.

Sau khi nén, công cụ sẽ hiển thị dung lượng trước/sau để bạn thấy kết quả.

Mọi thao tác đều được thực hiện 100% trên máy tính của bạn, không có dữ liệu nào được tải lên máy chủ. An toàn và Bảo mật tuyệt đối.
